# RAG Eval Toolkit

A complete framework for evaluating and comparing RAG (Retrieval-Augmented Generation) architectures.

See [rag_eval_toolkit/](rag_eval_toolkit/) for full documentation.

## Quick Start

```bash
cd rag_eval_toolkit
pip install -e .

# Run an evaluation
rag-eval run -c corpus.jsonl -d questions.jsonl
```
